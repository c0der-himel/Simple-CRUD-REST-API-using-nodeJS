# Simple-CRUD-REST-API-using-nodeJS

> Full CRUD REST API using nodeJS with no framework

This is for learning purposes.

## Usage

```
# Install dependencies
npm install
yarn install

# Run server
npm start
yarn start
```

## Routes

```
GET      /api/courses
POST     /api/courses
GET      /api/courses/:id
PUT      /api/courses/:id
DELETE   /api/courses/:id
```
